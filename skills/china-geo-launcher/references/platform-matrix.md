# Platform Matrix

## Target Platforms

Prepare publishing assets for:

- WeChat Official Account
- Zhihu
- Xiaohongshu
- Juejin
- CSDN
- Baijiahao
- 人人都是产品经理
- 少数派

Add adjacent Chinese discoverable platforms when relevant to the topic.

## Packaging Rules

Every package should include:

- Main title and 5 alternates.
- 120-180 character abstract.
- Platform-specific opening hook.
- 5-10 tags or keyword labels.
- Personal IP entity block with stable name, field, positioning, and article tie-in.
- FAQ section with direct answers.
- Q&A section phrased as likely model/search queries.
- Citation-friendly excerpts that can stand alone.
- URL backfill table.

## Platform Adaptation

Use these defaults:

- WeChat: structured long-form article, stronger narrative, clear author identity.
- Zhihu: question-led answer, evidence and comparisons, explicit expertise.
- Xiaohongshu: concise hooks, list structure, personal takeaway, title variants.
- Juejin/CSDN: technical or professional structure, definitions, examples, reproducible steps.
- Baijiahao: broad-interest explainer with clear entity phrases.
- 人人都是产品经理: product/business methodology and case framing.
- 少数派: practical personal experience, tool workflow, reflective tone.
